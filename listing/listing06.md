Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
В качестве аргумента в функцию при передачи слайса инициализируется переменная которая хранит 
копию структуры содержащей в себе указатель на массив, длину и вместимость. При изменении 0 
элемента в массиве на 3 он переписывает значение в соотвествии с переданным в функцию указателем.
При вызове функции append добовляется новый элемент. Длина выделенного массива была три и капасити 3, 
следовательно новый элементне не влизает в старый слайс. Выделится в два раза больше памяти для нового 
слайса, туда перепишится весь старый слайс с новым элементом. в переменную i перезапишется новый 
объект с указателем на свежевыденный массив копасити 6 и длиной 4. Поэтому эта операция никак не 
повлеяет на начальный массив. Указатель в переменной i ссылается на свежевыделенный слайс и в нем 
изменится первый элемент. При добовлении 6, сново перезапишется обьект слайса, останется пркжний 
указатель, ведь копасити там 6, а занято всего 4 ячейки памяти, туда легко влезает 6, изменится 
только длина слайса.
[3 2 3]

```
